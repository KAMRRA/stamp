🔥 Firebase 설정
처음 한 번만 설정하면 돼요
1
console.firebase.google.com 접속 → 프로젝트 추가
2
프로젝트 생성 후 웹 앱 추가 (</>) → 앱 닉네임 입력 → 등록
3
좌측 빌드 → Realtime Database → 데이터베이스 만들기 → 테스트 모드로 시작
4
프로젝트 설정 (⚙️) → 내 앱 → firebaseConfig 복사 → 아래 붙여넣기
const firebaseConfig = { apiKey: "AIza...", authDomain: "myapp.firebaseapp.com", databaseURL: "https://myapp-default-rtdb...", projectId: "myapp", ... };
